👁️ Структурна подібність (SSIM)

SSIM(x,y) = [l(x,y)]^α × [c(x,y)]^β × [s(x,y)]^γ

💡 Яскравість

l(x,y) = (2μₓμᵧ + C₁)/(μₓ² + μᵧ² + C₁)

🎨 Контраст

c(x,y) = (2σₓσᵧ + C₂)/(σₓ² + σᵧ² + C₂)

🏗️ Структура

s(x,y) = (σₓᵧ + C₃)/(σₓσᵧ + C₃)

🎵 Метрики для аудіо

PEAQ

Perceptual Evaluation of Audio Quality

ODG: 0 до -4
  • 0: непомітні зміни
  • -1: помітні, не дратівливі
  • -2: злегка дратівливі
  • -3: дратівливі
  • -4: дуже дратівливі

Segmental SNR

SSNR = (1/M) × Σ 10log₁₀(SNR_segment)

Переваги:

  • Краще для мови
  • Менша чутливість до пауз
  • Локальна оцінка якості

📊 Log-Spectral Distance (LSD)

LSD = √((1/N) × Σ [10log₁₀(Pₓ/Pᵧ)]²)

Вимірює відстань між спектрами оригіналу та модифікованого сигналу

📈 Статистичні метрики для стегоаналізу

KL-дивергенція

D_KL(P||Q) = Σ P(i)log(P(i)/Q(i))
  • D_KL ≥ 0
  • D_KL = 0 тільки якщо P = Q
  • Несиметрична

Відстань Бгаттачарія

D_B = -ln(Σ √(P(i)×Q(i)))
  • D_B ∈ [0, ∞)
  • D_B = 0 для ідентичних
  • Симетрична

🚚 Earth Mover's Distance (EMD)

Мінімальна "робота" для перетворення одного розподілу в інший

  • Враховує відстань між бінами
  • Стійкість до малих зсувів
  • Інтуїтивна інтерпретація

📦 Метрики ємності та ефективності

Коефіцієнт вбудовування

ER = |M| / |C|

Типові значення:

  • LSB: 1-15%
  • Адаптивні: 0.1-5%
  • Робастні: 0.01-1%

Bits Per Pixel (bpp)

bpp = |M| / (W × H)

Класифікація:

  • Низька: < 0.1 bpp
  • Середня: 0.1-0.5 bpp
  • Висока: > 0.5 bpp

📋 Підсумки та вибір метрик

🎯 Вибір метрик для застосувань

🕵️ Непомітна стеганографія

  • SSIM > 0.95
  • PSNR > 40 dB
  • KL < 0.1

💪 Робастна стеганографія

  • BER < 0.01
  • NC > 0.7
  • SR > 80%

📦 Велика ємність

  • Максимізація ER
  • PSNR > 35 dB
  • Оптимізація bpp

🔍 Стегоаналіз

  • KL-дивергенція
  • χ² статистика
  • EMD
Ключовий принцип: Комбінуйте об'єктивні та перцептуальні метрики

💡 Практичні рекомендації

  • SSIM краще корелює з людським сприйняттям ніж PSNR
  • Для кожного застосування потрібен свій набір метрик
  • Баланс між метриками визначається цілями системи
  • Регулярне тестування на реальних даних
